## Projects
  ### E-commerce Marketing Funnel Analysis
  
  **Link:** [`E-commerce Marketing Funnel Analysis`](Ecommerce-Marketing-Funnel/README.md)
  
  **Tools:** SQL and Power BI
  
  **Skill used:** 
  - *SQL:* Joins, Aggregate Functions, GROUP BY, CASE Statements, Date & Time Filtering, Data Type Casting, NULL Handling, CREATE TABLE / CREATE VIEW.
  - *Power BI:* Data import, calculated measures, KPI visualisation, funnel and bar charts, and basic formatting.
  
  **Description:**
  This project analyses a multi-CSV e-commerce dataset to construct a sequential signup-to-purchase marketing funnel. Using cleaned transactional and event data, the     analysis tracks how users progress through key funnel stages and identifies where drop-offs occur after signup.
  
   **Results:**
   - The signup-to-purchase conversion rate is approximately 51%.
   - The largest user drop-off occurs between the desire and action stages, indicating friction at the final purchase step.
   - Earlier funnel stages show relatively higher retention, suggesting that user intent is established before checkout.
